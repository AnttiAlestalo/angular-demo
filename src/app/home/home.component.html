Welcome to Angular demo application!

This application shows how some common web application features could be build with Angular. This is a responsive Single Page Application.

Routing: Angular router
Internationalization: Angular Internationalization i18n. Not implemented.
Icons: Angular Font Awesome
Dropdown menus: Own code
AJAX: Angular HttpClient
Simple table with sorting and filtering: Own code
Form with different field types; text, select, radio button, textarea: Angular Template-driven Forms
Searchable select list: ng-select
Date picker: ng2-datepicker
Form validation: Angular Template-driven validation
Popup: Custom modal dialog
Charts: ng2-charts
